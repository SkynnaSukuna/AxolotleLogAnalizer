# 📋 LogAnalyzer

**Анализатор логов** — десктопное приложение для просмотра, поиска и анализа логов.
Работает полностью **на вашем компьютере**, ничего не отправляет в интернет (кроме AI-анализа).

Построено на **Tauri 2.0** — легковесный десктопный клиент без браузера.

---

## 🚀 Быстрый запуск (Windows)

### Требования

1. **Node.js** (LTS): https://nodejs.org/
2. **Rust**: https://rustup.rs/ (установщик сам всё настроит)

### Запуск (1 клик!)

Просто **дважды кликните** `start.bat` — скрипт сам проверит зависимости и запустит приложение.

### Вручную

```bash
npm install
npm run tauri:dev
```

### Сборка `.exe` установщика

```bash
npm run tauri:build
```

Готовый установщик будет в `src-tauri/target/release/bundle/nsis/`.

---

## 📖 Как пользоваться

### Первый запуск

При первом запуске вас встретит небольшое обучение из 4 шагов.
В приложении уже есть **3 демо-проекта** с тестовыми логами — можете сразу начать!

### Основные возможности

| Действие | Как сделать |
|---|---|
| **Открыть лог** | Нажмите на проект слева, затем на нужный лог |
| **Загрузить логи** | Нажмите «+ Загрузить» или перетащите файлы в панель слева |
| **Найти ошибки** | Включите вкладку «Ошибки» сверху лога |
| **Поиск по тексту** | Используйте строку поиска над логом |
| **Создать заметку** | Кликните правой кнопкой на ошибке → «Создать заметку» |
| **Закрепить ошибку** | Кликните правой кнопкой → «Закрепить ошибку» |
| **AI-анализ** | Нажмите «Анализировать» в правой панели (требуется API-ключ) |
| **Сменить тему** | В левой панели, в самом верху — 4 кнопки тем |

### Поддерживаемые файлы

`.log` `.txt` `.json` `.csv`

## ⚙️ Техническая информация

- **Стек:** Tauri 2.0 + React + TypeScript + Vite + Tailwind CSS + Dexie.js
- **Хранение данных:** IndexedDB (встроенный браузерный движок Tauri)
- **Оффлайн:** Всё работает без интернета (кроме AI-анализа)
- **Темы:** Light / Dark / Terminal / IDE Style
- **Размер:** ~5-10 MB (без браузера, в отличие от Electron)

### Команды для разработки

```bash
npm run dev          # Vite dev server (без Tauri)
npm run tauri:dev    # Tauri dev (полноценное окно)
npm run tauri:build  # Сборка .exe / .dmg
npm run build        # Сборка фронтенда
```

## ❓ Частые вопросы

**Пропали данные после обновления?**
Данные хранятся в IndexedDB. Они не пропадают, т.к. Tauri не чистит storage.

**Как сделать резервную копию заметок?**
В правой панели (База знаний) нажмите ⬇ — это скачает JSON-файл.
Для восстановления нажмите ⬆.

**Ошибка «link.exe not found» / «dlltool.exe not found»?**
Установите Visual Studio Build Tools с C++ workload или используйте `rustup default stable-msvc` после установки C++ тулчейна.
