**Добро пожаловать в AxolotleLogAnalyzer**!

Это приложение поможет вам анализировать логи, находить ошибки и сохранять решения. Сделаное Skynna.

---

**AxolotleLogAnalyzer** — это не просто просмотрщик логов, а полноценная экосистема для хранения, глубокого анализа и накопления базы знаний по ранее решённым проблемам.

Весь процесс разработки и дев-лог проекта можно найти в Telegram-канале: https://t.me/tut_obitaet_skynna

---

## Главные преимущества
* Удобное хранение истории — группировка логов по проектам без хаоса в файлах.
* AI-анализ критических ошибок — умный сканер отсеивает шум и находит реальные причины сбоев.
* Поиск похожих случаев — мгновенный поиск по прошлым логам и готовым решениям.
* Система умных заметок — база знаний, которая сама подсказывает решения при повторных ошибках.
* Автоматический запрос недостающих логов — ИИ подскажет, если для анализа нужен crash-report.

## Разработчикам
В репозитории по пути `\app\dev` находятся полезные bat-файлы для автоматизации рабочих процессов:
* `check.bat` — запускает команду `tsc --noEmit + vite build` для быстрой и строгой проверки сборки.
* `clean-cache.bat` — полностью очищает директории сборки и кэша: `cargo target`, `node_modules/.vite` и `dist`.
* `start-dev.bat` — запускает среду окружения `tauri dev` в отдельном окне.
* `uninstall.bat` — скрипт для полного удаления программы.

## Основной функционал

### Просмотр и визуализация логов
* Подсветка уровней логирования: Автоматическое выделение строк по уровню важности в стиле Mclo.gs (INFO — зелёный, WARN — жёлтый, ERROR/FATAL/EXCEPTION — красный).
* Связанные строки в один клик: При обнаружении ошибки подсвечивается весь блок (stack trace, сообщения «до» и «после», а также вложенные исключения).
* Отдельные панели: Вкладки для быстрой навигации: «Все сообщения», «Ошибки», «Предупреждения». Клик по записи переносит в нужное место лога.
* Быстрые фильтры: Моментальное скрытие и отображение нужных типов записей без перезагрузки файла.

### Проекты и организация логов
* Папки проектов: Создание изолированных пространств (например, Survival, SkyBlock, Lobby, Test Server) со своей историей. Функция полезна для саппортов, работающих с большим количеством серверов.
* Отдельные чаты логов: Каждый лог — это чат с датой создания, где сохраняются сам файл, история общения с ИИ, результаты анализа и заметки пользователя. Все данные сохраняются локально на компьютере.
* Поиск похожих случаев: ИИ ищет похожие ошибки и stack trace среди всех прошлых логов и выводит дату, проект, предполагаемую причину и решение, если оно было сохранено.

### Система заметок и навигация
* Умные заметки: Привязка к ошибке причины (например, «Пустой config.yml») и решения («Перегенерировать конфиг»). При повторном появлении похожей проблемы заметка отображается автоматически.
* Панель закреплений: Сворачиваемая боковая панель с разделами «Закреплённые ошибки», «Заметки» и «Важные места» для быстрого перехода к нужному месту в логе.

### AI-анализ
* AI Critical Scan: Умное сканирование критичности. ИИ отличает рядовой ERROR от фатальных проблем (OutOfMemoryError, краш сервера, повреждение мира, ошибки запуска, ошибки БД, критические ошибки ядра). ИИ самостоятельно определяет уровень опасности.
* AI Вердикт: Наглядная карточка после анализа (Вероятная причина, Уверенность в %, Срочность, Требуются ли дополнительные логи) с кнопкой «Почему ИИ так считает?» для просмотра обоснования.
* Определение недостающих логов: ИИ анализирует контекст файла и автоматически сообщает, если для точной диагностики требуются дополнительные файлы (crash-report, latest.log, debug.log).

### Казуальные функции
* Темы оформления: Мгновенное переключение между темами «Светлая», «Тёмная», «Терминал» и «IDE Style».
* Метки логов: Назначение логам тегов («Срочно», «В работе», «Решено», «Отложено») для удобной сортировки и фильтрации.
* Значок «Решено»: Возможность установить статус решения для лога. После этого он помечается специальной иконкой и отображается в списке решённых проблем.
